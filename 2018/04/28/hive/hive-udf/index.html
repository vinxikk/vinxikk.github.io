大写转小写

  1. 创建UpperToLowerCase类，继承UDF，重写evaluate方法

  2. 打包上传到hadoop集群上

  3. 将jar包放到hive的classpath下

    0: jdbc:hive2://hadoop001:10086> add jar /home/vinx/jars/hive.jar;

  4. 创建临时函数，指定完整类名（包名+类名）

    0: jdbc:hive2://hadoop001:10086> create temporary function tolower as 'com.ivinx.hive.UpperToLowerCase';

  5. 使用临时函数

    select tolower('HELLO SPARK');
