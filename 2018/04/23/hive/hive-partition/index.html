业务场景：

参数设置：

  1. set hive.exec.dynamic.partition=true;（默认true，表示开启动态分区）
  2. set hive.exec.dynamic.partition.mode=nonstrict;（strict表示至少需要指定一个静态分区，nonstrict表示不需要指定静态分区）
