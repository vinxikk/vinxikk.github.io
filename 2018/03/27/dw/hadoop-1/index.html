HDFS主从架构

NameNode 名称节点，主节点，读写请求先经过它

DataNode 数据节点，从节点，存储数据，检索数据

SecondaryNameNode 第二名称节点，h+1 checkpoint检查点机制

大数据组件基本都是主从架构。

HBase（读写请求不经过HMaster进程）。

HDFS常用命令

hadoop fs == hdfs dfs

常用命令如下：

命令 描述
hadoop fs -mkdir 创建文件夹
hadoop fs -put 上传文件至HDFS
hadoop fs -get 下载文件至Client
hadoop fs -cat 查看文件内容
hadoop fs -rm 删除文件
hadoop fs -ls 查看指定目录下的所有文件
