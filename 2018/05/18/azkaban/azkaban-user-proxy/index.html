Azkaban用户代理

Azkaban可以代理其他linux用户执行命令，通过代理用户模式可以实现Hadoop的权限控制。

编译execute-as-user.c文件

在azkaban源码目录下，cd az-exec-util/src/main/c/

执行编译命令：gcc execute-as-user.c -o execute-as-user

得到execute-as-user，后面会用到。

用户代理配置

进入exec server目录：cd /home/vinx/app/azkaban-exec-server

新建文件夹：mkdir extlib

使用root用户，mv execute-as-user /home/vinx/app/azkaban-exec-server/extlib，并且修改权限：chmod 6050 /home/vinx/app/azkaban-exec-server/extlib/execute-as-user
