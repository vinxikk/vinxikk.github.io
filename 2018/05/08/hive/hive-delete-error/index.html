Hive中delete时抛错：...update or delete... not support these operations.
2018-05-08|Hive
Word count: 165|Reading time: 1 min

delete error

在Hive中使用delete删除数据时抛以下错误：

Error: Error while compiling statement: FAILED: SemanticException [Error 10294]: Attempt to do update or delete using transaction manager that does not support these operations. (state=42000,code=10294)

错误原因：

update和delete都属于事务操作，Hive的行级修改，需要开启事务。

解决办法：

在hive-site.xml中，增加如下配置：

hive.support.concurrency – true

hive.enforce.bucketing – true

hive.exec.dynamic.partition.mode – nonstrict

hive.txn.manager –org.apache.hadoop.hive.ql.lockmgr.DbTxnManager

hive.compactor.initiator.on – true

hive.compactor.worker.threads – 1

hive.in.test - true

保存后，重启Hive服务。
