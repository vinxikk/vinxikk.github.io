Linux vi命令

vi命令的三种模式

模拟流程：vi test.log进入命令模式，i键进入编辑模式，esc键退出编辑模式并进入命令模式，shift+:进入尾行模式，输入wq并回车保存并退出。

搜索

shift+:进入尾行模式后，/keyword，回车后自动匹配，N键寻找下一个

设置行号

进入尾行模式：

  • set nu 显示行号
  • set nonu 取消行号显示

清空文件内容

  • cat /dev/null > test.log
  • echo ‘’ > test.log 存在1个字节，慎用

命令模式常用快捷键

  • dd 删除当前行

  • dG 删除光标当前及以下的所有行

  • ndd 删除光标当前及以下的n行

  • gg 跳转到第一行的第一个字母

  • G 跳转到最后一行的第一个字母

  • shift + $ 行尾

  • gg + dG 清空文件
